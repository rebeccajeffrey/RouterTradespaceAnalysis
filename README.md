# Tradespace Analysis Tool

Interactive tradespace exploration and Pareto front visualization built with Streamlit.

## Features

- **Scatter plot** with configurable X/Y objectives
- **Pareto front** identification and highlighting
- **Parallel coordinates** for multi-dimensional comparison
- Upload your own CSV or explore with example data
- Color and size encoding for additional dimensions

## Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Deploy to Streamlit Community Cloud

1. Push this repo to GitHub
2. Go to [share.streamlit.io](https://share.streamlit.io)
3. Click "New app" and select this repository
4. Set the main file path to `app.py`
5. Click "Deploy"
